# PUPPETEER-BRIDGE

Bridge to print local html file to PDF

## Usage

```shell script
$ ./puppeteer --filepath=$PWD/sample/index.html \
              --pdfOutputPath=$PWD/sample/screen.pdf \
              --pdfMarginTop='30mm' \
              --pdfMarginLeft='20mm' \
              --pdfMarginRight='20mm' \
              --pdfMarginBottom='20mm' \
              --pdfLandscape=true
```
